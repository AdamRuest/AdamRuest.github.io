Password Generator

This project was done as an end of term project in my first programmation class. The goal was to be able to ask the user their first and last name and, from that, depending on if the initials are vowels or consonant. The project was done in TypeScript, and since it was the first class, we couldn't use "switch" statement so we had to have a bunch of "if" statement in order to know the layout of the generated password. Speaking of which, the layout looked like this: vowel, consonant, vowel, consonant, number from 1 to 99. The vowel and consonant were determined by the initals of the user, the first letter of each name filling up the first instant of that type of letter. The rest that weren't filled up were randomly generated.

This project, being the last project we did in TypeScript, didn't teach me anything in the language. Although we had never worked with the ASCII character table before, we didn't have to do the convertion ourself, since our teacher provided us wiht a function doing just that. Although the project in itself didn't teach me anything, the situation did. I was on a real time crunch to finish this project since i only had 3 days to do it. This as taught me that when i need to finish a project on a date, I can't afford to slackoff and need to work hard to finish it in time.

Like i mentionned above, the only thing that I didn't know how to do was the ASCII table random letter generator, but, fortunatly, our teacher was kind enough to provide us with the solution to that problem. Even if he gave us the solution, I wanted to understand how it worked, but couldn't at the time. Now that i did the C++ class thopugh, i can pretty easily understand this function and realise that i should have been able to understand it at the time, only thing was that i didn't do any research on the ASCII table chart since i didn't know how it was called. Other than that, the project was smooth sailing and was wuite easy to complete.
